# Nexus Streetwear Website

A professional, modern website for Nexus Streetwear - a premium quality clothing brand.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Black & White Theme**: Clean, modern aesthetic matching your brand identity
- **Product Showcase**: Display up to 6 products with images, names, prices, and "Add to Cart" buttons
- **About Section**: Highlights your brand values and commitment to quality
- **Contact Form**: Allows customers to get in touch easily
- **Social Media Links**: Instagram, Twitter, and Facebook integration
- **Smooth Navigation**: Fixed navigation bar with smooth scrolling to sections

## File Structure

```
NexusStreetwear/
├── index.html          # Main HTML file
├── styles.css          # All styling and responsive design
└── README.md          # This file
```

## How to Use

1. **Open the website**: Simply double-click `index.html` or open it in any web browser
2. **Add your product images**: Replace the placeholder image references (product1.jpg, product2.jpg, etc.) with your actual product images
3. **Customize content**: Edit the HTML file to update product names, prices, and descriptions
4. **Update contact info**: Change email, phone number, and social media links in the contact section

## Adding Your Product Images

To add your images:

1. Place your product images in the same folder as index.html
2. Name them: product1.jpg, product2.jpg, product3.jpg, product4.jpg, product5.jpg, product6.jpg
   - OR update the image src attributes in index.html to match your image filenames

## Customization Tips

- **Colors**: The black and white theme can be adjusted in styles.css
- **Products**: Add or remove product cards by copying/pasting the product-card div in index.html
- **Text**: All text content can be edited directly in index.html
- **Fonts**: Change the font-family in styles.css to use different fonts
